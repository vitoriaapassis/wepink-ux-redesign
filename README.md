# 🌸 Wepink UX/UI Redesign

## 🇧🇷 Português
Projeto de redesign UX/UI desenvolvido no Figma com foco em experiência do usuário, usabilidade, acessibilidade e arquitetura da informação para e-commerce. O projeto inclui prototipação de alta fidelidade, interações, fluxo de navegação, responsividade e validações de interface, buscando proporcionar uma experiência moderna, intuitiva e centrada no usuário.

### ✨ Funcionalidades
- Protótipo interativo
- Fluxo de navegação
- Sistema de favoritos
- Carrinho de compras
- Validação de formulários
- Responsividade
- Microinterações

### 🛠 Ferramentas
- Figma
- GitHub

## 🇺🇸 English
UX/UI redesign project developed in Figma focused on user experience, usability, accessibility, and information architecture for e-commerce. The project includes high-fidelity prototyping, interactions, navigation flow, responsiveness, and interface validations, aiming to deliver a modern, intuitive, and user-centered experience.

### ✨ Features
- Interactive prototype
- Navigation flow
- Favorites system
- Shopping cart
- Form validation
- Responsive design
- Microinteractions
